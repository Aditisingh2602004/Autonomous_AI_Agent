# Frontend Login Instructions

Frontend URL:
http://localhost:5173/login

Step 1: Open Login Page
Open the frontend login page in a browser.

Step 2: Fill Credentials
Use credentials from environment variables:
- USERNAME
- PASSWORD

Step 3: Submit Login Form
Fill these selectors:
- input[name="username"]
- input[name="password"]
- button[type="submit"]

Step 4: Verify Success
After submitting, wait for the UI to finish loading and confirm login success.

Rules:
- Use the frontend UI login only
- Do not call backend login unless explicitly asked
- Do not create or modify any data
